# Bridge Scores Dashboard

Interactive dashboard for the virtual bridge score history.

Live site:

https://virtual-bridge-explorer.netlify.app/

## Files

- `index.html` - dashboard markup
- `styles.css` - visual styling
- `script.js` - filtering, charts, year summaries, efficiency index
- `data.json` - exported bridge score data used by the dashboard
- `netlify.toml` - Netlify static-site publish settings

## Netlify

This is a static site. Netlify should publish the repository root with no build command.
